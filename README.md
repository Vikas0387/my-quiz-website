#  Quiz Game Website

An interactive **Quiz Game Website** built using **HTML**, **CSS**, and **JavaScript**.  
Users can attempt a quiz, answer multiple-choice questions, and view their score with animated results.


##  Features

-  **Home Page:** Clean landing page with navigation links.  
-  **Quiz Guide Popup:** Displays rules and instructions before starting.  
-  **Interactive Quiz:**  
  - One question displayed at a time  
  - Multiple-choice radio buttons  
  - Next/Submit button functionality  
-  **Result Section:**  
  - Shows score with circular progress animation  
  - Options to retry the quiz or return to the home page  
-  **Responsive Design:** Works smoothly on all devices.  
-  **Smooth animations** for popup, quiz transitions, and results.
